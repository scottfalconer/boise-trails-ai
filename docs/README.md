## Phone Field Packet

`field-packet/index.html` is the phone-first view for field testing. It is
generated by:

```bash
python years/2026/scripts/export_mobile_field_packet.py
```

The packet contains one GPX per runnable parked-start outing and skips manual
route-design holds. Each phone card includes a car-to-car cue list: park/start,
access from the car to the route, official segments, connector moves, and return
to car. It must not contain exact home-origin details or private local paths.
