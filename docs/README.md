## Phone Field Packet

`field-packet/index.html` is the installable phone-first PWA for field testing.
It is generated by:

```bash
python years/2026/scripts/export_mobile_field_packet.py
```

The packet contains one GPX per runnable parked-start outing, an all-GPX zip,
web app manifest, service worker, and app icons. It skips manual route-design
holds. Each phone card includes a car-to-car cue list: park/start, access from
the car to the route, official segments, connector moves, and return to car.

On iPhone, open the field packet in Safari and use Share -> Add to Home Screen.
After the first full load, the service worker caches the packet and GPX files for
minimal offline backup. Completion/hide state is local to the device browser. It
must not contain exact home-origin details or private local paths.
