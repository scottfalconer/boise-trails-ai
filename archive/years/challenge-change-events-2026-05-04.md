# Challenge Change Events

Created: 2026-05-04

This file tracks year-specific closures, construction, and organizer adjustments that can explain target-count and mileage changes across Boise Trails Challenge years.

Source note: the events below come from user-provided excerpts from past Boise Trails Challenge emails. I found matching trail names in local planner artifacts, but not the original email source text in this repo. Treat these as documented working evidence until the original emails are archived.

## Why This Matters

Annual target differences are not automatically model errors. The challenge can change during the event because of fire, construction, Forest Service closures, or access constraints. For model-comparison work:

- Use the final public history target for final completion math.
- Use the local planner-era official file to reconstruct what last year's model optimized against.
- Treat mismatches as possible challenge-state drift unless segment-level evidence proves a parser or planner bug.

## Events

### 2024-06-18: Heroes Trail Access And Bogus Basin XC Construction

Affected trails: Heroes Trail, Bogus Basin XC trails, Lower Deer Point, Around the Mountain.

Organizer email excerpts provided by the user said the hospital parking lot at the south end of Heroes Trail was being redone until June 22, so athletes either needed to wait or do an out-and-back from Elephant Rock. The same note described significant Bogus Basin XC trail changes, including the closure of Lower Deer Point.

Planning impact: do not assume normal trailhead access or normal Bogus XC connector topology during the affected window. This may not change final segment counts, but it changes practical route feasibility.

### 2024-06-24: Polecat Gulch Fire Closure

Affected trails: Polecat Gulch / Polecat Loop.

Organizer email excerpts provided by the user said a fire closed Polecat Gulch. Organizers had not fully removed it from the challenge at that point, but temporarily marked Polecat complete for athletes who had completed everything else while they waited to see if it would reopen.

Planning impact: public final completion math may include organizer-side administrative credit or removal logic that differs from a static pre-event official segment file.

### 2025-06-25: Ridge Road And Mahalo Access Closure

Affected trails: Ridge Road, Mahalo Trail.

Organizer email excerpts provided by the user said the Forest Service closed Ridge Road, limiting access to Mahalo, and acknowledged that participants had to adapt.

Planning impact: this is a plausible explanation for the 2025 mismatch between the local planner file and public final history target. The local file may reflect an initial/static official set while public final completion math reflects post-closure organizer adjustments.
