# Archive

This folder holds prior Boise Trails planning work so each new challenge year can start from a clean active workspace.

## Contents

- `legacy-root-2025/` - the pre-2026 root workspace: old code, tests, configs, data, docs, scripts, notebooks, generated GPX/images, research bundle, local caches, and the old README.
- `years/` - historical year folders and public-history summaries for 2018-2025.

## Boundaries

- Do not use archived code as current 2026 implementation without intentionally copying or porting it into the active workspace.
- Do not put credentials or OAuth tokens in archive bundles.
- Archived paths are for retrospective/model-comparison evidence and recovery.
