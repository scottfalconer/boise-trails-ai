# 2025 Legacy Root Archive

This directory preserves the planner implementation that previously lived at
the repository root during the 2025 planning cycle.

It is archived for retrospective comparison only. Current 2026 implementation
work should happen under `years/2026/`, not by editing this legacy package.

The committed archive intentionally excludes local runtime artifacts and
private or heavy inputs:

- local virtual environments, caches, bytecode, and generated package metadata
- private dashboard/API pulls
- personal GPX results
- generated route-output dumps and plan images
- large OSM/DEM files

Legacy official 2025 input snapshots are preserved under
`archive/years/2025/inputs/official/`.
