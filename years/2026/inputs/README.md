# 2026 Inputs

This folder separates current 2026 inputs from old baseline files.

## Expected Inputs

- `official/`: official 2026 segment list, rules, challenge dates, directional requirements, and target totals.
- `personal/`: user preferences, availability, expected route durations, pace assumptions, injury/fatigue constraints, and manual progress notes.
- `strava/`: 2026 Strava exports or API pulls needed for segment completion and activity evidence.
- `open-data/`: refreshed connector trail network, OSM roads, and elevation data if needed.

## Current State

No 2026 official challenge source has been loaded yet. Do not copy the 2025 official file here unless it is explicitly marked as a schema fixture or temporary comparison source.
